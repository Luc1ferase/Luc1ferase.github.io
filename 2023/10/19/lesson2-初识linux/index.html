linux简介

Linux是一种自由和开放源码的类UNIX操作系统。该操作系统的内核由林纳斯·托瓦兹在1991年10月5日首次发布，再加上用户空间的应用程序之后，就成为了Linux操作系统。Linux也是自由软件和开放源代码软件发展中最著名的例子。只要遵循GNU 通用公共许可证（GPL），任何个人和机构都可以自由地使用Linux的所有底层源代码，也可以自由地修改和再发布。大多数Linux系统还包括像提供GUI的X Window之类的程序。除了一部分专家之外，大多数人都是直接使用Linux 发行版，而不是自己选择每一样组件或自行设置。

Linux严格来说是单指操作系统的内核，因操作系统中包含了许多用户图形接口和其他实用工具。如今Linux常用来指基于Linux的完整操作系统，内核则改以Linux内核称之。由于这些支持用户空间的系统工具和库主要由理查德·斯托曼于1983年发起的GNU计划提供，自由软件基金会提议将其组合系统命名为GNU/Linux，但Linux不属于GNU计划，这个名称并没有得到社群的一致认同。

Linux最初是作为支持英特尔x86架构的个人电脑的一个自由操作系统。目前Linux已经被移植到更多的计算机硬件平台，远远超出其他任何操作系统。Linux可以运行在服务器和其他大型平台之上，如大型计算机和超级计算机。世界上500个最快的超级计算机已100％运行Linux发行版或变种。Linux也广泛应用在嵌入式系统上，如手机（Mobile Phone）、平板电脑（Tablet）、路由器（Router）、电视（TV）和电子游戏机等。在移动设备上广泛使用的Android操作系统就是建立在Linux内核之上。

通常情况下，Linux被打包成供个人计算机和服务器使用的Linux发行版，一些流行的主流Linux发布版，包括Debian（及其派生版本Ubuntu、Linux Mint）、Fedora（及其相关版本Red Hat Enterprise Linux）和openSUSE等。Linux发行版包含Linux内核和支撑内核的实用程序和库，通常还带有大量可以满足各类需求的应用程序。个人计算机使用的Linux发行版通常包含X Window和一个相应的桌面环境，如GNOME或KDE。桌面Linux操作系统常用的应用程序，包括Firefox网页浏览器、LibreOffice办公软件、GIMP图像处理工具等。由于Linux是自由软件，任何人都可以创建一个符合自己需求的Linux发行版。

kali linux介绍

Kali Linux是基于Debian的Linux发行版，设计用于数字鉴识和渗透测试。由Offensive Security公司维护和资助。

Kali Linux内置了大约600个渗透测试程序（工具），包括Armitage（图形化网络攻击管理工具）、Nmap（端口扫描工具）、Wireshark（数据包分析器）、Metasploit（渗透测试框架）、John the Ripper（密码破解器）、sqlmap（自动SQL注入和数据库接管工具）、Aircrack-ng（用于渗透测试无线局域网的软件包）、Burp suite以及OWASP ZAP网络应用安全扫描器等。

下载

首先下载两个需要的文件：

VMware workstation pro 17
kali-linux-2023.3-vmware-amd64.7z
网络不好的同学可以直接下载钉钉群中的文件。

安装

  1. 打开下载好的VMware-workstation-full-17.0.0-20800274.exe按照正常安装软件的流程安装即可，值得注意的是到这一步时记得勾选上增强型键盘驱动程序，忘记了也没关系，后续还可以安装。

    打开后输入激活码：NZ4RR-FTK5H-H81C1-Q30QH-1V2LA，此时的软件应该永久激活了。
    如果第一次错过了输入激活码，可以在上方的工具栏中点击帮助-输入许可证秘钥再次输入激活码激活永久版。

  2. 解压kali-linux-2023.3-vmware-amd64.7z到你认为合适的目录，最好是在一个空间充足的盘下建一个专门用来存放虚拟机的文件夹，因为后续如果你喜欢上linux可能会尝试各种发行版，每个发行版都要占用数G存储空间，笔者此时的虚拟机文件夹已达200G+。如果你所使用的解压缩程序不支持解压.7z文件，我推荐一个开源的解压缩程序NanaZip，可以支持大部分的压缩格式文件，网络问题无法打开Github的同学可以直接在钉钉群中下载40174MouriNaruto.NanaZip_2.0.450.0_gnj4mf6z9tkrc (1).msixbundle并安装。
    解压完成之后应该是如下图形式的一个文件夹，到这里就先不用管这个文件了。

  3. 打开VMware Workstation Pro，工具栏中选择文件-打开，找到步骤2中解压好的那个文件夹，打开之后其中应该只有一个可以打开的文件，后缀为.vmx，在开启虚拟机之前可以按照自己电脑的配置和存储空间情况自定义一下虚拟机的硬件情况，配置好之后点击开启此虚拟机，第一次开启可能会耗费较长时间，耐心等待。

  4. 一段时间之后，你应该已经来到了kali linux的登录界面，首次登录的用户名和密码均为kali。

  5. 进入到桌面后按快捷键Ctrl+Alt+t打开一个终端(terminal)，此后你使用linux的操作都要在这个黑色框框中完成。注意，linux中的复制粘贴为Ctrl+Shift+C、Ctrl+Shift+V。
    输入命令sudo su并输入密码切换到超级用户，注意，linux中输入密码是没有显示的，不要以为没输上。再执行命令bash <(curl -sSL https://linuxmirrors.cn/main.sh)输入数字快捷更换软件源，我一般喜欢使用科大的源，有ustc lug专门维护，稳定性啥的各方面都不错，也可以执行bash <(curl -sSL https://linuxmirrors.cn/main.sh) --edu 用咱们学校自己的镜像源。

执行完成后提示是否跳过更新软件包? [Y/n]输入Y（你也可以输入n自动更新，但是想带你们走一下常规流程。）

安装完系统之后要干的几件事

  1. 获取需要更新的软件包
$ sudo apt update      
[sudo] password for kali: 
Hit:1 https://mirrors.ustc.edu.cn/kali kali-rolling InRelease
Reading package lists... Done         
Building dependency tree... Done
Reading state information... Done
847 packages can be upgraded. Run 'apt list --upgradable' to see them.

可以解释为：sudo 以root执行，apt 包管理器，update同步系统上的软件包列表。
最下边那行则是：有847个可更新的软件包，输入apt list --upgradable来查看有哪些。

  1. 更新软件包
$ sudo apt upgrade
[sudo] password for kali: 
Reading package lists... Done
Building dependency tree... Done
Reading state information... Done
Calculating upgrade... Done
The following packages were automatically installed and are no longer required:
此处会列出一堆软件包的名字

Y

问你是否要更新，Y/n。

  1. 更改密码
    $ passwd更改用户kali的密码，输入一下原密码，输入两次新密码即可。

切换到root用户，$ sudo su，输入# passwd更改root用户的密码，输入一下原密码，输入两次新密码即可。

命令之前的”$”、”#”不需要输入，代表不同的权限

  1. 安装你需要的软件
    以安装neofetch举例
    $ sudo apt install neofetch即可

名词解释

sudo：以超级用户执行后边的命令，经由 sudo 所执行的指令就像是 root 亲自执行。

apt：高级打包工具（英语：Advanced Packaging Tools，缩写为APT）是Debian及其派生的Linux软件包管理器。APT可以自动下载，配置，安装二进制或者源代码格式的软件包，因此简化了Unix系统上管理软件的过程。

passwd：password缩写

开源：开放源代码（英语：Open Source）是一种在电脑程序出品前和开发中提供其内部代码的做法。

镜像源：镜像源就是把官方的源做一个镜像，你可以在这下载软件。 比如kali官方源在国外，下载软件可能会很慢，这时候就需要换成国内的镜像源。

Linux发行版：Linux 发行版（英语：Linux distribution或distro，也被叫做GNU/Linux 发行版），为一般用户预先集成好的Linux操作系统及各种应用软件。一般用户不需要重新编译，在直接安装之后，只需要小幅度更改设置就可以使用，通常以软件包管理系统来进行应用软件的管理。Linux发行版通常包含了包括桌面环境、办公包、媒体播放器、数据库等应用软件。这些操作系统通常由Linux内核、以及来自GNU计划的大量的函式库，和基于X Window或者Wayland的图形界面。有些发行版考虑到容量大小而没有预装 X Window，而使用更加轻量级的软件，如：BusyBox、musl或uClibc-ng。现在有超过300个Linux发行版（Linux发行版列表）。大部分都正处于活跃的开发中，不断地改进。
