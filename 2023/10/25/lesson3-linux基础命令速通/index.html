lesson3-linux基础命令速通

2023/10/25 给NSS新人的教程 Linux

前言

本节课程较为填鸭式，但是不难，多为记忆内容，大胆的去操作练习即可。当你需要什么操作的时候尽量不要第一时间想着用图形界面操作，先去想想或者查查用什么命令可以完成。因为在我们以后接触实战时大多数系统是没有图形界面的，所以要尽快适应命令操作。

基础用法

如果没有特别指明，本篇中的指令格式为：

序号. 指令
该指令的全写
$ 该指令的用法
其他注意事项

  1. ssh
    $ ssh luciferase@127.0.0.1
    同学们如果不理解的话，可以想想windows上的远程桌面(rdp协议)，ssh可以粗略的理解成linux的远程桌面协议，用于从你现在这台机器向‘@’之后的ip地址的目标机器发送连接请求。默认端口是22，指令后边可以加上-p来指定端口，比如$ ssh -p 2222 luciferase@hostname。

  2. ls
    list files，列出文件。
    $ ls
    列出当前目录下的文件和文件夹
    $ ls -a可以列出所有文件和文件夹，包括隐藏文件。
    $ ls -l可以列出文件的详细信息，包括文件的权限、所有者、大小、创建时间等等。
    $ ls -al可以列出所有文件的详细信息，包括隐藏文件。

  3. pwd
    print working directory，打印当前工作目录。
    $ pwd

  4. cd
    change directory，改变目录。
    $ cd /home/luciferase
    只输入$ cd会回到当前用户的home目录
    $ cd ..会回到上一级目录
    $ cd ../..会回到上两级目录，以此类推。
    $ cd -会回到上一次所在的目录。

  5. mkdir
    make directory，创建目录。
    $ mkdir test会在当前目录下创建一个名为test的文件夹。

  6. touch
    $ touch test.txt会在当前目录下创建一个名为test.txt的文件。
    $ touch welcome to nss会在当前目录下创造三个文件，分别为welcome、to、nss。
    $ touch nss{1..10}.txt会在当前目录下创造10个文件，分别为”nss1.txt、nss2.txt、nss3.txt、nss4.txt、nss5.txt、nss6.txt、nss7.txt、nss8.txt、nss9.txt、nss10.txt。”

  7. echo
    $ echo "hello world"会在终端输出hello world。
    $ echo "hello world" > test.txt会在当前目录下创建一个名为test.txt的文件，并将hello world写入该文件。
    $ echo "hello world" >> test.txt会在当前目录下创建一个名为test.txt的文件，并将hello world追加到该文件的末尾。

  8. nano
    $ nano test.txt会用nano打开test.txt文件(如果当前目录下没有test.txt的话。它会先在缓存中写入，最后问你要不要保存到名为test.txt文件中)，
    写完之后Ctrl+x，Y，回车保存。nano是一个文本编辑器，可以用来编辑文本文件。

  9. vim
    $ vim test.txt用法暂不详细讲解，遇到问题可以问我。想学的同学可以参考Just Vim It学习
    比nano更有逼格的文本编辑器，但是上手难度也更高，不太建议新手使用，熟练之后装逼效果拉满。

  10. cat
    concatenate 连锁
    $ cat test.txt会在终端输出test.txt文件的内容。
    $ cat test.txt > test2.txt会将test.txt文件的内容写入test2.txt文件中。
    $ cat test.txt >> test2.txt会将test.txt文件的内容追加到test2.txt文件的末尾。

  11. shred
    $ shred test.txt会将test.txt文件重复覆盖，使得即使是昂贵的硬件探测仪器也难以将数据复原,相当于碎纸机的作用。

  12. cp
    Copy file 复制文件
    $ cp test1.txt /home/testdir/将当前文件夹下的test1.txt文件复制到/home/testdir文件夹下。

  13. mv
    move 移动文件
    $ mv test1.txt /home/luciferase将当前文件夹下的test1.txt文件移动到/home/luciferase文件夹下

  14. clear
    $ clear
    顾名思义，清空当前终端的内容。

  15. rm(高危命令)
    remove 删除文件
    $ rm test.txt删除当前文件夹下的test.txt文件。
    $ rm -r testdir删除当前文件夹下的testdir文件夹，-r表示递归删除，即删除文件夹下的所有文件和文件夹。
    $ rm -rf testdir表示无提示地强制递归删除文件。
    在你按下回车键时一定要再三确定要删除的目录或者文件路径是否正确，删除操作难以逆转。

  16. rmdir
    remove directory 删除文件夹
    $ rmdir testdir删除当前文件夹下的testdir文件夹，只能删除空文件夹。
    当文件夹不为空时，可以使用$ rm -r testdir尝试递归删除。

  17. whoami
    字面意思，告诉你当前的用户身份
    $ whoami

  18. adduser
    字面意思，添加用户
    $ adduser testuser添加一个名为testuser的用户，按照提示输入密码即可。

  19. su
    switch user 切换用户
    $ su testuser切换到testuser用户，按照提示输入密码即可。
    $ su - testuser切换到testuser用户，并且切换到testuser用户的home目录下。
    $ exit退出当前用户，回到上一个用户。

  20. passwd
    $ passwd修改当前用户的密码。
    $ passwd testuser修改testuser用户的密码。
    一般要用sudo执行。

  21. man
    manual 手册
    $ man apt查看apt的手册（用法），一般为英文，可以自行黏贴翻译，按q退出。

  22. whatis
    字面意思，查看指令的简介
    $ whatis apt查看apt的简介。

  23. whereis
    字面意思，查看指令的位置
    $ whereis apt查看所有apt的位置。

  24. which
    基本只在$PATH路径中搜索，查找范围最小，查找速度快。默认只返回第一个匹配的文件路径。
    $ which apt查看apt的位置。
    科普：环境变量($PATH)

  25. wget
    GNU Wget（常简称为Wget）是一个在网络上进行下载的简单而强大的自由软件，目前它支持通过HTTP、HTTPS，以及FTP这三个最常见的TCP/IP协议协议下载。
    $ wget https://www.baidu.com下载百度首页到当前目录下。

  26. curl
    基于网络协议，对指定URL进行网络传输。
    $ curl https://www.baidu.com下载百度首页到当前目录下。
    $ curl https://www.baidu.com > 1.html下载百度的首页存储到文件1.html中

科普：wget和cURL的区别

  1. zip
    压缩
    $ zip testzip.zip 1.txt

  2. unzip
    解压缩
    $ unzip testzip.zip

  3. cmp
    compare 比较两个文件的不同
    $ cmp test1.txt test2.txt

  4. diff
    Difference 输出确切的两个文件的不同点
    $ diff test1.txt test2.txt

  5. chmod(高危命令)
    change mode 控制用户对文件的权限的命令
    当你当前的用户对某文件无法进行操作时，可以尝试使用chmod命令修改文件的权限。
    详见chmod与文件调用权限
    $ chmod +x xxx.sh

  6. chown(高危命令)
    change owner 设置文件所有者和文件关联组的命令。
    $ chown luciferase /home/luciferase/test.txt
    将/home/luciferase目录下的test.txt文件所有者设置成luciferase用户。
    这个命令通常需要root权限执行

  7. ifconfig
    interface config 网络接口配置
    $ ifconfig查看网络接口的配置信息

  8. ip address
    $ ip address查看网络接口的配置信息

  9. grep
    Global Regular Expression Print，全局正则表达式版本，用于搜索文本并输出匹配的行。
    $ grep "hello" test.txt在test.txt文件中搜索hello字符串，并输出匹配的行。
    正则表达式是什么
    ip address | grep "eth0" | grep "inet"
    管道符 “|” 的作用：将两个命令隔开，管道符左边命令的输出就会作为管道符右边命令的输入。
    这行命令是将ip address的结果传递给grep “eth0”的输入，再将ip address | grep "eth0"的结果传递给grep "inet"作为输入，最后输出匹配的行。

  10. ping
    $ ping www.baidu.comping百度的域名，查看网络是否通畅。
    $ ping -c 5 www.baidu.comping百度的域名5次，取平均值。参数-c代表count

  11. netstat
    network status 网络详情
    $ netstat查看当前的网络状态，包括连接、端口等

  12. ufw
    $ sudo ufw allow 22防火墙打开22端口
    $ sudo ufw enable起用防火墙
    $ sudo ufw status查看当前的防火墙详情

  13. uname
    $ uname -a查看当前系统内核版本(可以依据此查询是否存在内核漏洞)

  14. neofetch(与uname作用类似，但是更漂亮)
    $ sudo apt install neofetch安装neofetch
    $ neofetch查看当前系统的信息

  15. free
    $ free查看当前系统剩余多少内存

  16. df
    disk free 查看硬盘空间状态
    $ df -H可以得到一个清晰的列表

  17. ps
    Processes Status 进程详情
    $ ps
    $ ps -aux
    -a选项表示显示所有用户的进程，而不仅仅是当前用户的进程。
    -u选项表示显示进程的详细信息，包括用户名、启动时间、CPU使用率等。
    因此，ps -aux命令将显示所有用户的进程，并提供详细信息。这是一种常用的查看系统中运行进程的方法。

  18. htop
    $ htop
    相当于windows系统上的任务管理器

  19. kill(高危指令)
    $ kill 1234杀死进程号为1234的进程
    相当于windows系统上的任务管理器中的结束进程

  20. systemctl(高危指令)
    $ sudo systemctl start firewalld 起用firewalld服务
    $ sudo systemctl stop firewalld 停用firewalld服务

  21. history
    $ history查看历史命令

  22. reboot&shutdown
    $ reboot重启系统
    $ shutdown -h now立即关机
    $ shutdown -h 1010分钟后关机
